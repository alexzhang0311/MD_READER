# 使用指南

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问 [http://localhost:5000](http://localhost:5000)

## 默认账户

| 用户名 | 密码 |
|--------|------|
| admin  | admin123 |

> ⚠️ 请在生产环境中修改默认密码！

## 配置说明

编辑 `config.py` 文件来自定义设置：

```python
# Markdown 文件目录
MARKDOWN_DIR = '/path/to/your/markdown/files'

# 服务端口
PORT = 5000

# 密钥（请修改为随机字符串）
SECRET_KEY = 'your-secret-key'
```

## 环境变量

也可以通过环境变量来配置：

- `MD_READER_DIR` — Markdown 文件目录路径
- `MD_READER_PORT` — 服务端口号
- `MD_READER_SECRET` — 会话密钥
- `MD_READER_DEBUG` — 是否开启调试模式（true/false）
