# Markdown 语法参考

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 强调

- **粗体**: `**粗体**`
- *斜体*: `*斜体*`
- ~~删除线~~: `~~删除线~~`

## 列表

### 无序列表

- 项目一
- 项目二
  - 子项目 A
  - 子项目 B
- 项目三

### 有序列表

1. 第一步
2. 第二步
3. 第三步

### 任务列表

- [x] 已完成任务
- [x] 另一个已完成任务
- [ ] 未完成任务

## 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.png)
```

## 代码

行内代码: `code`

代码块:

```python
print("Hello, World!")
```

## 表格

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |

## 分割线

---

## 引用

> 这是一段引用文字
>
> 可以多行
